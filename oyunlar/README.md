# Oyunlar

Bu klasor Hakorocks Studio oyun kaynaklarini tutar.

Canli sitede oyunlar su yollarla acilir:

- `/oyunlar/annenden-kac/`
- `/oyunlar/bardak/`
- `/oyunlar/essiz-zindan/`
- `/oyunlar/skeleton-wars/`
- `/oyunlar/vale/`
- `/oyunlar/robot-avcisi/`

Yeni oyun eklerken kaynak klasoru burada tutulur, ana sayfadaki oyun karti `src/main.js` icinden eklenir ve gerekirse `scripts/build-games.mjs` build akisi guncellenir.
