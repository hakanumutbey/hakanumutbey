# minik

Bu klasor parti modu sunucusu icin.

- Oyuncu bilgisayardan parti olusturur.
- Sunucu 4 haneli parti kodu verir.
- Katilmak isteyen oyuncu bu kodu girer.
- Herkes hazir olunca parti baslar.
- Party lideri herkes hazirken oyunu baslat komutunu da gonderebilir.
- Partiyi kuran oyuncu baskalarini partiden atabilir.
- Party uyeleri yazili sohbet edebilir.
- En fazla 3 parti acilir.
- Her partide en fazla 5 oyuncu olur.
- Oyun dosyalarini da ayni sunucu yayinlar.

Calistirma:

```bash
npm install
npm start
```

Sonra tarayicida `http://localhost:3000` acilir.
