# Projeler

Bu klasor yeni web projeleri, denemeler ve uygulamalar icin ayrildi.

Yeni bir proje eklerken:

- Projeyi `projeler/proje-adi/` altina koy.
- Ana sitede gorunmesi gerekiyorsa `src/main.js` icindeki `projects` listesine ekle.
- Tarayicida calisan bir proje ise build surecine eklenmesi gerekip gerekmedigini kontrol et.
